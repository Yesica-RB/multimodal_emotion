# Multimodal Emotion Recognition in Travel Social Media Posts

Proyecto universitario — Grado en Ingeniería en Sistemas Inteligentes (UIE)

## Descripción
Sistema multimodal que predice la emoción (positiva, negativa, neutral)
en posts de Twitter sobre viajes combinando texto e imagen.
Integra NLP clásico, BERT, Computer Vision clásica y CNN con transfer learning
dentro de un framework de agente racional.

## Asignaturas cubiertas
- Speech and Natural Language Processing (Isaac González)
- Advanced Machine Learning (Alejandro Esteban)
- Computer Vision (Rubén Ferreiroa)
- Intelligent Systems (Yago Fontenla)

## Dataset
MVSA-Single — 5.000 pares imagen+texto de Twitter con etiquetas de sentimiento.
